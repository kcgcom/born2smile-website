---
name: 콘텐츠 수정 전 사전 검토 필수
description: 블로그 포스트 등 콘텐츠 변경 시 실행 전 반드시 수정 내용을 먼저 보여주고 확인받아야 함
type: feedback
---

콘텐츠(블로그 포스트, 텍스트) 수정 시 계획 단계에서 바로 실행하지 말고, 변경할 내용을 먼저 사용자에게 보여준 뒤 확인을 받고 실행한다.

**Why:** 사용자가 내용을 직접 검토하고 싶어하며, 확인 없이 바로 실행하면 의도와 다른 방향으로 진행될 수 있음.

**How to apply:** 블로그 포스트 blocks/content 수정, 문체 다듬기, 구조 개편 등 모든 콘텐츠 변경 작업에 적용. 코드/설정 변경은 해당 없음.
