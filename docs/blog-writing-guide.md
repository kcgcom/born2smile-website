# 블로그 작성 가이드

> 블로그 포스트 작성 시 참조하는 가이드입니다. 프로젝트 개요 및 기술 정보는 `CLAUDE.md`를 참고하세요.

## 브랜드 보이스

친절하고 따뜻하면서도 전문성을 잃지 않는 동네 치과 원장님 톤. 환자에게 직접 설명하듯 쓰되, 가볍지 않게.

서울본치과 블로그의 브랜드 보이스는 세 가지 축으로 구성된다:

| 키워드 | 의미 | 글에서의 표현 |
|--------|------|-------------|
| **정직함** | "꼭 필요한 치료만" | 과장·공포 유발 금지. "큰일납니다" → "초기에 치료하면 간단하게 해결됩니다" |
| **따뜻함** | "평생 주치의" | 환자 불안 먼저 인정 → 안심. "걱정되시죠? 정상적인 반응입니다" |
| **전문성** | "정석대로 하는 치료" | 구체적 수치·기간 사용. "일시적으로" → "보통 2~3일" |

## 문체 원칙

- 기본 어미: `~이에요`, `~해요`, `~거든요`, `~드려요` (부드러운 존댓말)
- 정보·정의 문장은 `~합니다/됩니다` 허용, 공감·권유는 `~해요/세요` 선호
- **같은 길이 문장 3개 이상 연속 금지** — 복문→단문→복문 리듬 유지
- 모호한 시간·정도 표현 금지 — 구체적 수치로 표현한다

## 인간미 있는 리듬 기법

딱딱하지 않고 사람이 쓴 글처럼 느껴지게 하는 세 가지 기법을 적극 활용한다.

1. **짧은 단독 문장 삽입** — 긴 설명 뒤에 짧게 끊어준다.
   예: "그게 핵심이에요."

2. **질문형 전환** — 새 소주제를 시작할 때 독자의 궁금증을 질문으로 대신한다.
   예: "그렇다면 어떻게 알 수 있을까요?"

3. **경험·대화 인용** — 치과 현장에서 자주 듣는 말이나 상황을 짧게 넣는다.
   예: "'안 아파서 그냥 뒀어요'라고 하시는 분들 정말 많아요."

## 피해야 할 표현

- **마케팅 과장어**: ~~투자~~, ~~혁신적인~~, ~~획기적인~~ → `방법`, `효과적인` 등
- **영어 전문 용어 단독 사용**: ~~리테이너~~, ~~Essix~~ → 유지장치
- **공포 마케팅**: ~~방치하면 치아를 잃습니다~~ → `초기에 치료하면 간단합니다`
- **단정 표현**: ~~반드시~~ → `대부분의 경우`, `일반적으로`

## 글 구조

| 요소 | 기준 |
|------|------|
| 제목(title) | 환자 시점의 질문·증상·오해를 후킹 문구로 |
| 부제(subtitle) | `[주제] + [가치/범위]`, 한 줄에 들어올 만큼 짧게 |
| 요약(excerpt) | 2~3문장. `[환자 상황] → [이 글이 제공하는 정보]` + 핵심 키워드 포함 |
| 본문 섹션 | **5개**, 한 섹션 = 한 아이디어. 한 호흡에 읽힐 분량으로 — 너무 짧으면 설명 부족, 너무 길면 스크롤 이탈 |

## 섹션 전개 패턴

| 섹션 | 환자 감정 | 글의 역할 |
|------|----------|----------|
| 1. 도입 | 불안·궁금 | "나도 그래!" 공감 → 안심 또는 정의 제시 |
| 2. 원인·배경 | 이해하고 싶음 | 왜 그런지 원인·메커니즘을 쉬운 비유로 설명 |
| 3. 분류·상세 | 더 알고 싶음 | 유형·단계·비교 등 전문 정보를 환자 눈높이로 |
| 4. 실천 | 뭘 해야 하지? | "이렇게 하면 됩니다" 구체적 행동 안내 |
| 5. 마무리 | 안심·신뢰 | 요약 → 치과 방문 조건 → 서울본치과 멘트 |

- 치과 개념은 환자에게 낯선 경우가 많다. 어려운 메커니즘이 나오는 섹션에서 비유나 일상적 비교로 풀어준다. 단, 의학적으로 부정확한 비유는 피한다.
- 마무리 멘트 유형: 상담 안내형 / 조건부 방문형 / 예방 철학형 / 안심 마무리형
- 서울본치과 언급은 **글에서 다룬 내용과 직접 연결**되어야 한다. 연결점이 없으면 넣지 않는다.

**자연스러운 연결 예시:**
- 스케일링 주기 글 → "개인 맞춤 주기를 안내해 드리고 있어요"
- 임플란트 vs 브릿지 비교 글 → "어떤 방향이 더 맞는지 함께 이야기 나눠드려요"
- 사랑니 발치 기준 글 → "발치가 필요한지 솔직하게 알려드려요"

**억지 삽입 예시 (피해야 함):**
- 본문 내용과 무관한 병원 연락처나 광고성 문구로 끝맺기
- "서울본치과에서 최고의 치료를 받아보세요" 류의 과장
- 이탤릭 한 줄로 뚝 끊어서 광고처럼 붙이기

## 전문 용어 처리

**3단계 표기 원칙** — 괄호 표기는 SEO에 유리하지만, 남용하면 글이 장황해지고 전문성을 오히려 해친다. 다음 기준으로 판단한다.

**A. 괄호 없이 바로 사용** — 환자 대부분이 아는 용어
예: 치석, 발치, 임플란트, 크라운, 레진, 치주염, 근관치료

**B. 괄호에 대체 키워드 병기 (SEO 이중 노출)** — 첫 등장 시만
예: `충치(치아우식증)`, `신경치료(근관 치료)`, `스케일링(치석 제거)`, `시린 이(지각 과민)`

**C. 괄호에 짧은 설명** — 환자에게 생소한 비일상 전문 용어
예: `법랑질(치아의 가장 바깥층)`, `골유착(뼈와 결합하는 과정)`, `차폐막(뼈이식 부위를 보호하는 막)`

이후 등장부터는 A·B·C 모두 용어만 사용. 영어는 괄호 보충만 허용, 본문에서 단독 사용 금지.

## SEO / GEO

### 검색 의도 구분

글을 쓰기 전에 주제의 검색 의도를 먼저 파악한다. 의도에 따라 제목·구조·강조점이 달라진다.

| 의도 | 예시 질문 | 글의 중심 |
|------|-----------|----------|
| **정보형** | "스케일링이 뭔가요?" | 원인·메커니즘·개념 설명 |
| **행동형** | "스케일링 받아야 하나요?" | 판단 기준·실천 안내 |
| **비교형** | "임플란트 vs 브릿지 뭐가 나아요?" | 장단점·상황별 선택 기준 |

### 소제목 키워드 (SEO)

H2 소제목에 환자 검색 표현이 자연스럽게 포함되도록 한다.

- 나쁜 예: "치아 관리의 중요성"
- 좋은 예: "스케일링, 얼마나 자주 받아야 하나요?"

### E-E-A-T 신호

치과·의료는 Google이 엄격히 평가하는 YMYL 콘텐츠다. 과장 없이 경험과 전문성이 느껴지는 표현을 글 어딘가에 자연스럽게 포함한다.

예: "실제 진료에서 자주 확인하는 부분이에요" / "치과에서 권장하는 기준은 ~" / "많은 분들이 오해하시는 부분인데요"

### 환자 검색 표현 매핑

제목·도입부·소제목에 구어체 검색어 포함:

- **시린 이**: "찬물 마시면 이가 시려요", "양치할 때 이가 시림", "시린이 원인"
- **잇몸**: "잇몸에서 피가 나요", "잇몸이 붓고 아파요", "잇몸이 내려앉았어요", "잇몸 색이 변했어요"
- **충치**: "이가 아파요", "이가 까맣게 변했어요", "충치 초기 증상", "충치 자연치유"
- **교정**: "치아교정 기간", "투명교정 후기", "교정 후 유지장치", "덧니 교정"
- **임플란트**: "임플란트 통증", "임플란트 수명", "임플란트 vs 브릿지", "임플란트 후 관리"
- **사랑니**: "사랑니 꼭 빼야 하나요", "사랑니 발치 후 관리", "사랑니 통증"
- **보철**: "크라운 종류", "크라운 수명", "앞니 깨졌을 때", "금니 vs 지르코니아"
- **예방**: "스케일링 주기", "올바른 양치법", "치실 사용법", "아이 충치 예방"
- **틀니**: "틀니 관리법", "틀니 불편함", "임플란트 vs 틀니", "부분 틀니 비용"
- **보존치료**: "레진 수명", "인레이 vs 레진", "이차 충치", "충치 치료 과정"
- **신경치료**: "신경치료 통증", "신경치료 후 크라운", "신경치료 기간", "신경치료 비용"
- **소아치과**: "아이 첫 치과", "유치 충치 치료", "실란트 시기", "아이 불소 도포"
