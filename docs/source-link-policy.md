# 출처 링크 정책

서울본치과의 블로그, FAQ, 진료 안내 페이지에서 외부 출처 링크를 추가할 때 따르는 운영 기준입니다.

## 목적

- 의료 정보의 신뢰도를 높인다.
- 잘못된 링크, 과장된 인용, 오래된 제도 정보로 인한 신뢰 하락을 막는다.
- 콘텐츠 작성자와 개발자가 같은 기준으로 출처를 고를 수 있게 한다.

## 기본 원칙

1. 출처는 많게 넣는 것보다 정확하게 넣는 것이 우선이다.
2. 의료·제도 정보는 가능하면 1차 출처만 사용한다.
3. 병원 철학, 상담 방식, 경험적 설명에는 출처 링크를 강제로 붙이지 않는다.
4. 숫자, 연령, 보험, 급여, 제도 문장은 특히 보수적으로 다룬다.
5. 외부 링크는 실제 문장과 직접 연결되는 경우에만 넣는다.

## 출처 분류

### A. 권장 출처

바로 사용 가능한 1차 출처다.

- 정부/공공기관
  - `mohw.go.kr`
  - `nhis.or.kr`
  - `hira.or.kr`
  - `kdca.go.kr`
- 공식 학회/전문기관
  - `ada.org`
  - `mouthhealthy.org`
  - `aapd.org`
  - `iadt-dentaltrauma.org`
- 원문 논문/리뷰
  - `pubmed.ncbi.nlm.nih.gov`
  - `pmc.ncbi.nlm.nih.gov`
- 공신력 있는 공공 의료 안내
  - `nhs.uk`

### B. 조건부 허용 출처

환자 안내나 일반 설명에는 사용할 수 있지만, 보험/급여/법적 기준 문장에는 사용하지 않는다.

- 대학병원 환자 안내 페이지
  - 예: `cuh.nhs.uk`, `guysandstthomas.nhs.uk`
- 대형 비영리 의료기관의 환자 교육 페이지

사용 조건:

- 치료 후 관리
- 시술 전후 주의사항
- 환자용 일반 설명
- 특정 제도나 법적 기준이 아닌 일반 의료 정보

### C. 비허용 출처

- 일반 블로그
- 상업적 마케팅 사이트
- 경쟁 치과 사이트
- 출처 불명 요약글
- 커뮤니티 게시글
- AI가 생성한 정리 페이지

## 문장 유형별 기준

### 보험 / 연령 / 급여 / 제도

가장 엄격하게 다룬다.

- 원칙: 한국 공공기관/공식 제도 출처만 사용
- 본문에는 세부 규정을 길게 박제하지 않는다
- “최신 기준은 변동될 수 있음”을 전제로 쓴다

좋은 예:

- `건강보험 적용 여부는 연령과 현재 상태에 따라 달라질 수 있어 상담 시 최신 기준으로 함께 확인해드립니다.`
- `최신 스케일링 건강보험 적용 기준 보기`

피해야 할 예:

- `만 65세 이상은 정확히 ...입니다`
- `본인부담률은 항상 ...입니다`

### 일반 치료 설명 / 환자 안내

- 공식 학회
- NHS
- 조건부 허용 대학병원 환자 안내
- 필요 시 논문/리뷰

### 연구 근거 / 수치 / 비교

- PubMed / PMC
- 공식 가이드라인
- 학회 문서

## 문장 작성 원칙

1. 한 문장 한 근거를 원칙으로 한다.
2. 근거가 약한 문장에 단정 표현을 쓰지 않는다.
3. 다음 표현을 우선 사용한다.
   - `일반적으로`
   - `현재 기준으로`
   - `개인 상태에 따라 달라질 수 있습니다`
   - `최신 기준 확인이 필요합니다`

## 링크 노출 방식

- 링크 텍스트는 “어디 링크인지”보다 “무엇을 확인하는 링크인지”가 드러나야 한다.

좋은 예:

- `건강보험 적용 기준 보기`
- `임플란트 수술 후 관리 안내 보기`
- `외상 치아 응급 처치 가이드 보기`

피해야 할 예:

- `여기`
- `출처`
- `링크`

## 운영 절차

출처 보강 작업은 아래 순서로 진행한다.

1. 문장 유형 분류
   - 보험/급여/연령/제도
   - 일반 환자 안내
   - 연구 근거/수치
2. 출처 후보 수집
3. A/B/C 분류 기준에 따라 선별
4. 링크 대상이 실제 문장을 직접 뒷받침하는지 확인
5. 200 응답 여부 확인
6. 본문 표현이 과장되지 않았는지 재검토

## 검수 체크리스트

- 공식 기관 또는 허용된 출처인가
- 내가 쓴 문장과 실제 출처 내용이 정확히 맞는가
- 한국 제도 문장인데 해외 출처를 쓰고 있지 않은가
- 제도/급여 문장에 수치 박제가 과하지 않은가
- 링크가 열리는가
- 문장 끝의 표현이 단정적이지 않은가
