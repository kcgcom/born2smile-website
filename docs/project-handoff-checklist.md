## B. 오너가 직접 제공/결정해야 하는 것

### 1. 환경변수 및 시크릿 (필수)

- [ ] **Kakao Maps App Key** — 로컬 개발 시 `.env.local`에 `NEXT_PUBLIC_KAKAO_MAP_APP_KEY` 값 제공
- [ ] **Firebase API Key** — `NEXT_PUBLIC_FIREBASE_API_KEY` 값 제공 (좋아요 기능 테스트용)
- [ ] 현재 `.env.local` 파일이 존재하지 않아 로컬에서 지도/Firestore 기능 테스트 불가

### 2. 비어있는 SNS 링크 (`lib/constants.ts` → `LINKS`)

| 키 | 용도 | 상태 |
|----|------|------|
| `kakaoChannel` | Contact 페이지 카카오 상담 버튼 + Footer 아이콘 | 빈 문자열 |
| `instagram` | Footer 아이콘 | 빈 문자열 |
| `naverBlog` | Footer 아이콘 | 빈 문자열 |
| `naverMap` | Footer 아이콘 | 빈 문자열 |
| `kakaoMap` | 지도 연결 | 빈 문자열 |

- [ ] 위 SNS 계정/페이지가 있으면 URL 제공
- [ ] 없으면 "없다"고 확인 (향후 개설 계획 여부도 함께)
- 값을 채우면 Footer 아이콘, Contact 페이지 카카오 상담 버튼, JSON-LD `sameAs`에 자동 반영됨

### 3. 비즈니스 결정이 필요한 TODO 항목

- [ ] **환자 후기 섹션** — `docs/homepage-improvement-ideas.md`에 아이디어 존재. 추가할지, 한다면 후기 콘텐츠 제공 필요
